# 四川省 · 사천성

Next.js로 만든 클래식 사천성(그림 맞추기) 게임입니다.

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 http://localhost:3000 접속

## 게임 방법

- **같은 그림**의 타일 두 개를 클릭하여 연결하면 제거됩니다.
- 연결 경로는 **최대 2번만 꺾을** 수 있습니다 (직선, L자, Z자 형태).
- 경로 위에 다른 타일이 있으면 연결할 수 없습니다.

## 배경 이미지

- **상단 50%**: `public/bg-xi-jinping.png` — 시진핑 주석님 사진 (바둑판 격자 형식)
- **하단 50%**: `public/bg-china-flag.png` — 중국 국기 (바둑판 격자 형식)

## BGM

BGM을 사용하려면 `public/bgm.mp3` 파일을 추가하세요. 화면 우측 하단의 BGM 버튼으로 재생/일시정지, 음소거를 조절할 수 있습니다.

## 기능

- **힌트 (F1)**: 매칭 가능한 타일 쌍을 표시 (5회)
- **섞기 (F2)**: 남은 타일을 재배치 (3회)
- **재시작**: 새 게임 시작

## 기술 스택

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
